# Integer Functions

## Floors and ceilings
* [x] (floor) = 小於等於 x 的最大整數
  * ex: [3.8] = 3
* [x] (ceiling) = 大於等於 x 的最小整數
  * ex: [3.8] = 4
* {x} (the fractional part of x) = 取 x 的小數
  * ex: {3.8} = 0.8
