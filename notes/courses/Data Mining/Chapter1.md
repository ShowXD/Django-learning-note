# 資料探勘簡介
## 什麼是數據挖掘
* 從大量數據中提取有趣的（重要的、 隱含的、以前未知的和可能有用的）模式或知識

## 可以挖掘什麼
* 泛化
* 發現新 Pattern
* 分類
* 聚類分析
* 異常值分析
* 時間與排序
* 結構與網路分析

### 發現新 Pattern(Pattern Discovery)
### 分類(Classification)
### 聚類分析(Cluster Analysis)
